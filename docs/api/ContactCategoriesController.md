# ContactCategoriesController

Path: `FinanceManager.Web.Controllers.ContactCategoriesController`

Purpose:
- Manage contact categories for organizing contacts.

Key endpoints
- `GET /api/contact-categories` - list
- `POST /api/contact-categories` - create
- `PUT /api/contact-categories/{id}` - update
- `DELETE /api/contact-categories/{id}` - delete

Notes
- Name uniqueness per owner enforced.
