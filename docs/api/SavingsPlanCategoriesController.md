# SavingsPlanCategoriesController

Path: `FinanceManager.Web.Controllers.SavingsPlanCategoriesController`

Purpose:
- Manage categories used for organizing savings plans.

Key endpoints
- `GET /api/savings-plan-categories` - list
- `POST /api/savings-plan-categories` - create
- `PUT /api/savings-plan-categories/{id}` - update
- `DELETE /api/savings-plan-categories/{id}` - delete

Notes
- Only owner-scoped items allowed.
