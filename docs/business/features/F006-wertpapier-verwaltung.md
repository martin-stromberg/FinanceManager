# F006 – Wertpapier-Verwaltung

## Einleitung

Die Wertpapier-Verwaltung ermöglicht es Ihnen, Aktien, Fonds und andere Wertpapiere zu verwalten. Sie können Wertpapiere erfassen, kategorisieren und den Bestand verfolgend. Dies ist Teil einer umfassenden Vermögensbetrachtung.

## Wer nutzt es?

**Finanzverwalter und Vermögensmanager** nutzen diese Funktion, um ihre Wertpapierbestände zu dokumentieren. Dies ist besonders wichtig für die Vermögensplanung und Risikoabschätzung.

## Schritt-für-Schritt-Anleitung

### Neues Wertpapier erfassen

1. Sie navigieren zu **Wertpapiere**.
2. Sie klicken **Neues Wertpapier**.
3. Sie tragen folgende Informationen ein:
   - **Name/ISIN** (z.B. "Siemens AG", "DE0007236101")
   - **Typ** (z.B. "Aktie", "Fonds")
   - **Kategorie** (z.B. "DAX-Aktie", "Rohstoff-Fonds")
   - **Menge** (Anzahl der Anteile/Aktien)
   - **Kaufpreis** (optional)
4. Sie klicken **Speichern**.

### Wertpapierbestand aktualisieren

1. Sie öffnen ein Wertpapier.
2. Sie klicken **Bearbeiten**.
3. Sie aktualisieren die **Menge** und **Kaufpreis**.
4. Sie klicken **Speichern**.

### Wertpapier verwalten

1. Sie öffnen ein Wertpapier.
2. Sie sehen die Detailinformationen und die aktuellen Preise.
3. Sie können das Wertpapier in verschiedenen Kategorien organisieren.

## Beispiel

Sie besitzen 50 Anteile der Siemens AG und 20 Anteile eines Immobilienfonds:

- **Wertpapier 1**: "Siemens AG", ISIN "DE0007236101", 50 Anteile, Kaufpreis 95 EUR
- **Wertpapier 2**: "Immobilien-Fonds XY", ISIN "DE0001234567", 20 Anteile, Kaufpreis 50 EUR

Sie erfassen beide in der Software und sehen den aktuellen Wert Ihrer Bestände.

## Was passiert im Hintergrund?

Die Software speichert die Wertpapierinformationen und kann die aktuellen Preise abrufen (siehe F007). Damit kann Sie den aktuellen Wert Ihres Portfolios berechnen.

## Häufige Fragen (FAQ)

**F: Was ist eine ISIN?**  
A: Eine ISIN ist die internationale Wertpapierkennnummer. Sie eindeutig identifiziert ein Wertpapier (z.B. DE0007236101).

**F: Kann ich den Kaufpreis nachträglich ändern?**  
A: Ja, Sie können alle Informationen jederzeit aktualisieren.

**F: Wie wird der aktuelle Wert berechnet?**  
A: Der aktuelle Wert = Menge × aktueller Preis. Die Preise werden automatisch abgerufen (F007).

**F: Kann ich Wertpapiere verkaufen?**  
A: Sie können die Menge aktualisieren, um einen Verkauf zu dokumentieren.

**F: Werden Dividenden erfasst?**  
A: Dividenden können als separate Transaktionen erfasst werden.
