# F012 – Kontakte

## Einleitung

Die Kontakte-Verwaltung ermöglicht es Ihnen, Geschäftspartner, Lieferanten und Kunden zu registrieren. Sie können Kontaktinformationen speichern, um später schnell darauf zuzugreifen. Kontakte werden oft mit Transaktionen verknüpft (z.B. Lieferant, dem Sie Geld schuldend).

## Wer nutzt es?

**Sachbearbeiter und Finanzverwalter** nutzen diese Funktion, um ihre Geschäftspartner im System zu erfassen. Dies unterstützt die Dokumentation von Geschäftsbeziehungen.

## Schritt-für-Schritt-Anleitung

### Neuen Kontakt erstellen

1. Sie navigieren zu **Kontakte**.
2. Sie klicken **Neuer Kontakt**.
3. Sie füllen folgende Informationen aus:
   - **Name** (Name der Person oder Firma)
   - **Kategorie** (z.B. "Lieferant", "Kunde", "Bank")
   - **E-Mail** (optional)
   - **Telefon** (optional)
   - **Adresse** (optional)
4. Sie klicken **Speichern**.

### Kontakt bearbeiten

1. Sie öffnen die Kontaktliste.
2. Sie klicken auf einen Kontakt.
3. Sie klicken **Bearbeiten**.
4. Sie aktualisieren die Informationen.
5. Sie klicken **Speichern**.

### Kontakt löschen

1. Sie öffnen die Detailseite eines Kontakts.
2. Sie klicken **Löschen**.
3. Der Kontakt wird entfernt (nur wenn keine Verbindungen bestehen).

## Beispiel

Sie arbeiten regelmäßig mit der Agentur "WebDesign Plus" zusammen. Sie erstellen einen Kontakt:

- **Name**: "WebDesign Plus GmbH"
- **Kategorie**: "Agentur"
- **E-Mail**: "info@webdesign-plus.de"
- **Telefon**: "+49 30 12345678"
- **Adresse**: "Musterstraße 1, 10115 Berlin"

Später können Sie Rechnungen dieser Agentur verknüpfen und schnell alle Transaktionen mit diesem Kontakt anzeigen.

## Was passiert im Hintergrund?

Die Software speichert die Kontaktinformationen in der Datenbank. Diese Daten können mit Transaktionen verknüpft werden, um die Nachverfolgung zu erleichtern.

## Häufige Fragen (FAQ)

**F: Kann ich Kontakte in Kategorien organisieren?**  
A: Ja, Sie können Kategorien wie "Lieferant", "Kunde" oder "Bank" erstellen und Kontakte zuordnen.

**F: Kann ich einen Kontakt als Selbst markieren?**  
A: Ja, es gibt eine spezielle Kategorie "Selbst" für Ihre eigene Organisation.

**F: Was passiert, wenn ich einen Kontakt lösche, der mit Transaktionen verknüpft ist?**  
A: Die Transaktionen bleiben erhalten, der Kontakt kann aber nicht gelöscht werden.

**F: Kann ich Kontakte exportieren?**  
A: Das hängt von den Einstellungen ab. Wenden Sie sich an Ihren Administrator.

**F: Kann ich Kontaktgruppen erstellen?**  
A: Ja, Sie können Kontakte in Gruppen organisieren.
