# F010 – Ersparnispläne

## Einleitung

Die Ersparnispläne helfen Ihnen, Sparziele zu definieren und zu verfolgen. Sie können festlegen, wie viel Sie pro Monat sparen möchten, und die Software zeigt Ihnen, ob Sie Ihre Ziele erreichen. Dies ist wichtig für finanzielle Vorsorge und Zielplanung.

## Wer nutzt es?

**Finanzverwalter und Vermögensplaner** nutzen diese Funktion, um Sparziele zu verwalten. Dies ist besonders wichtig für Notgroschen, Altersvorsorge oder Ersparnisse für spezifische Ziele.

## Schritt-für-Schritt-Anleitung

### Sparplan erstellen

1. Sie navigieren zu **Ersparnispläne** oder **Savings Plans**.
2. Sie klicken **Neuer Sparplan**.
3. Sie tragen folgende Informationen ein:
   - **Name des Sparziels** (z.B. "Notgroschen", "Urlaub 2024")
   - **Zielbeträge** (z.B. 10.000 EUR)
   - **Gewünschte Sparrate** (z.B. 500 EUR/Monat)
   - **Startdatum** und **Enddatum**
   - **Kategorie** (optional, z.B. "Sicherheit", "Freizeit")
4. Sie klicken **Speichern**.

### Sparplan-Fortschritt überwachen

1. Sie öffnen die **Ersparnispläne-Übersicht**.
2. Sie sehen für jeden Plan:
   - Zielbeträge
   - Bereits gesparrt
   - Verbleibender Betrag
   - Fortschrittsbalken
   - Erwartetes Enddatum basierend auf aktuellem Sparen

### Transaktionen zum Sparplan zuordnen

1. Sie erfassen eine Ausgabe (z.B. +500 EUR auf Sparkonto).
2. Sie öffnen die Ausgabenverwaltung.
3. Sie verknüpfen diese Transaktion mit dem Sparplan "Notgroschen".
4. Die Software addiert diese zum Sparplan-Fortschritt.

## Beispiel

Sie möchten ein Notgroschen von 10.000 EUR aufbauen. Sie erstellen einen Sparplan:

- **Name**: "Notgroschen"
- **Zielbeträge**: 10.000 EUR
- **Sparrate**: 500 EUR/Monat
- **Geplantes Enddatum**: 31.12.2024

Sie sparen konsequent 500 EUR pro Monat. Nach 12 Monaten:
- Geplant: 10.000 EUR (12 × 500)
- Tatsächlich: 9.800 EUR (Sie haben einen Monat ausgelassen)

Der Sparplan zeigt: "80% erreicht. Wird am 31.01.2025 abgeschlossen sein."

## Was passiert im Hintergrund?

Die Software verfolgt alle Transaktionen, die Sie dem Sparplan zuordnen. Sie berechnet den bisherigen Erfolg und extrapoliert, wann Sie Ihr Ziel erreichen werden, basierend auf dem bisherigen Sparprozess.

## Häufige Fragen (FAQ)

**F: Kann ich mehrere Sparziele gleichzeitig haben?**  
A: Ja, Sie können beliebig viele Ersparnispläne erstellen.

**F: Was passiert, wenn ich einen Sparplan nicht erreiche?**  
A: Es gibt keine Strafe, aber Sie erhalten einen Benachrichtigung über die Verzögerung.

**F: Kann ich einen Sparplan ändern?**  
A: Ja, Sie können Zielbeträge, Spatraten und Datumadjustieren.

**F: Werden Sparpläne automatisch gelöscht?**  
A: Nein, Sie können abgeschlossene Sparpläne archivieren oder löschen.

**F: Kann ich mehrere Konten für einen Sparplan nutzen?**  
A: Ja, alle Transaktionen mit diesem Sparplan werden zusammengefasst.
