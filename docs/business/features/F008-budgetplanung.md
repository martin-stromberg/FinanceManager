# F008 – Budgetplanung

## Einleitung

Die Budgetplanung ermöglicht es Ihnen, Ausgabengrenzen für verschiedene Kategorien festzulegen. Sie definieren, wie viel Sie pro Monat in jeder Kategorie maximal ausgeben möchten. Die Software überwacht dies und warnt Sie, wenn Sie das Budget überschreiten.

## Wer nutzt es?

**Finanzverwalter und Budget-Controller** nutzen diese Funktion, um Ausgaben zu kontrollieren und Kostenexplosionen zu verhindern. Dies ist wichtig für eine disziplinierte Finanzwirtschaft.

## Schritt-für-Schritt-Anleitung

### Budget erstellen

1. Sie navigieren zu **Budgetplanung** oder **Budgets**.
2. Sie klicken **Neues Budget**.
3. Sie wählen eine **Kategorie** (z.B. "Bürokosten", "Reisekosten").
4. Sie tragen den **Budgetbetrag** ein (z.B. 5.000 EUR für den Monat).
5. Sie wählen das **Budgetintervall** (Monat, Jahr, etc.).
6. Sie legen eine **Benachrichtigungsschwelle** fest (z.B. ab 80% warnen).
7. Sie klicken **Speichern**.

### Budget-Fortschritt überwachen

1. Sie öffnen das **Budget-Dashboard**.
2. Sie sehen für jedes Budget:
   - Budgetierter Betrag
   - Bereits ausgegebener Betrag
   - Verbleibender Betrag
   - Visueller Fortschrittsbalken
3. Budgets mit hohem Ausschöpfungsgrad werden farblich hervorgehoben.

### Budget anpassen

1. Sie öffnen das Budget-Dashboard.
2. Sie klicken auf ein Budget.
3. Sie klicken **Bearbeiten**.
4. Sie ändern die Budgetsumme oder die Schwellenwerte.
5. Sie klicken **Speichern**.

## Beispiel

Sie möchten Ihre Ausgaben kontrollieren und erstellen folgende Budgets:

**Budget 1:** "Bürokosten" – 5.000 EUR/Monat
- Benachrichtigung ab 80%
- Grenzwert: 4.000 EUR

**Budget 2:** "Reisekosten" – 2.000 EUR/Monat
- Benachrichtigung ab 75%

In diesem Monat haben Sie bereits:
- Bürokosten: 4.200 EUR (84% des Budgets) → **Warnung!**
- Reisekosten: 1.400 EUR (70% des Budgets) → OK

Sie sehen die Warnung und können die Bürokosten-Ausgaben überprüfen.

## Was passiert im Hintergrund?

Die Software berechnet monatlich (oder in einem anderen Intervall) die Summe aller Transaktionen pro Kategorie und vergleicht sie mit Ihrem festgelegten Budget. Wenn der Schwellenwert erreicht wird, wird eine Benachrichtigung generiert.

## Häufige Fragen (FAQ)

**F: Was passiert, wenn ich mein Budget überschreite?**  
A: Sie erhalten eine Warnung. Es gibt keine automatische Blockade, aber Sie werden benachrichtigt.

**F: Kann ich verschiedene Budgets für verschiedene Monate haben?**  
A: Ja, Sie können Budgets pro Monat oder per Regel definieren.

**F: Wie werden Budgets berechnet?**  
A: Nur Transaktionen mit der gewählten Kategorie werden gezählt.

**F: Kann ich mehrere Budgets pro Kategorie haben?**  
A: Dies hängt von der Systemkonfiguration ab. Normalerweise ein Budget pro Kategorie pro Zeitraum.

**F: Werden Budgets automatisch zurückgesetzt?**  
A: Ja, monatliche Budgets werden am 1. des Monats zurückgesetzt.
