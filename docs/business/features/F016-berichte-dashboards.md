# F016 – Berichte & Dashboards

## Einleitung

Die Berichte & Dashboards-Funktion bietet Ihnen einen Überblick über Ihre finanzielle Situation durch visuelle Diagramme und detaillierte Berichte. Sie können Einnahmen, Ausgaben und Vermögen analysieren und Trends erkennen.

## Wer nutzt es?

**Manager, Finanzverwalter und Stakeholder** nutzen diese Funktion, um finanzielle Leistungen zu analysieren und Entscheidungen zu treffen. Dies ist besonders wichtig für regelmäßige Geschäftsaudits.

## Schritt-für-Schritt-Anleitung

### Dashboard anzeigen

1. Sie navigieren zu **Berichte** oder **Dashboard**.
2. Sie sehen eine Übersichtseite mit Diagrammen und KPIs.
3. Sie sehen zum Beispiel:
   - Ausgaben pro Kategorie (Kreisdiagramm)
   - Einnahmen vs. Ausgaben (Balkendiagramm)
   - Vermögensübersicht (Vermögen, Schulden, Netto)

### Bericht erstellen

1. Sie klicken **Neuer Bericht**.
2. Sie wählen den **Berichtstyp** (z.B. "Jahresbericht", "Quartalsabrechnung").
3. Sie legen das **Zeitfenster** fest (z.B. Januar bis März 2024).
4. Sie wählen die **Kategorien**, die in den Bericht aufgenommen werden sollen.
5. Sie klicken **Bericht generieren**.

### Bericht exportieren

1. Sie öffnen einen erstellten Bericht.
2. Sie klicken **Exportieren** (z.B. als PDF oder Excel).
3. Der Bericht wird als Datei auf Ihren Computer heruntergeladen.

## Beispiel

Sie möchten einen Quartalsbericht für Q1 2024 erstellen:

1. Sie wählen **Neuer Bericht** → **Quartalsbericht**.
2. Sie legen fest: "01.01.2024 bis 31.03.2024".
3. Sie wählen alle Kategorien aus.
4. Sie klicken **Generieren**.

Der Bericht zeigt:
- Gesamteinnahmen: 50.000 EUR
- Gesamtausgaben: 35.000 EUR
- Gewinn: 15.000 EUR
- Ausgaben nach Kategorie (mit Diagrammen)

## Was passiert im Hintergrund?

Die Software aggregiert alle Transaktionen des Zeitraums, berechnet Summen und Prozentsätze und generiert visuellen Diagramme. Die Daten können in verschiedene Formate exportiert werden.

## Häufige Fragen (FAQ)

**F: Kann ich Favoriten-Berichte speichern?**  
A: Ja, Sie können häufig verwendete Berichte als Favoriten markieren.

**F: Welche Zeiträume sind möglich?**  
A: Sie können beliebige Zeiträume wählen (Tage, Wochen, Monate, Jahre).

**F: Können mehrere Benutzer den gleichen Bericht nutzen?**  
A: Ja, Berichte können geteilt werden (abhängig von den Berechtigungen).

**F: Wie oft werden die Daten aktualisiert?**  
A: Berichte werden beim Öffnen mit den aktuellen Daten aktualisiert.

**F: Kann ich Berichte automatisch generieren?**  
A: Dies hängt von den Einstellungen ab. Manuelle Generierung ist immer möglich.
