# F014 – Benutzereinstellungen

## Einleitung

Die Benutzereinstellungen ermöglichen es Ihnen, die FinanceManager-Anwendung an Ihre persönlichen Bedürfnisse anzupassen. Sie können Sprache, Zeitzone, Benachrichtigungspräferenzen und weitere Optionen konfigurieren.

## Wer nutzt es?

**Alle Nutzer der Anwendung** verwenden diese Funktion, um ihre persönlichen Einstellungen vorzunehmen. Dies geschieht in der Regel bei der Erstanmeldung.

## Schritt-für-Schritt-Anleitung

### Einstellungen öffnen

1. Sie klicken im **Menü** auf **Setup** oder **Einstellungen**.
2. Sie öffnen die Setup-Seite mit verschiedenen Tabs (Reiter).
3. Wählen Sie den Tab, den Sie bearbeiten möchten.

### Profil-Einstellungen (Tab: Profil)

1. Sie klicken auf den Tab **Profil**.
2. Sie können Ihre persönlichen Daten eingeben:
   - **Name**
   - **Zeitzone** (z.B. Europe/Berlin)
3. Sie klicken **Speichern**.

### Benachrichtigungen (Tab: Benachrichtigungen)

1. Sie klicken auf den Tab **Benachrichtigungen**.
2. Sie können folgende Optionen einstellen:
   - **Benachrichtigungen aktivieren** (ja/nein)
   - **Benachrichtigungszeit** (Uhrzeit für tägliche Benachrichtigungen)
   - **Benachrichtigungstypen** (welche Events sollen benachrichtigt werden?)
3. Sie klicken **Speichern**.

### Sicherheit (Tab: IP-Filter / Sicherheit)

1. Sie klicken auf den Tab **Sicherheit** oder **IP-Filter**.
2. Sie können IP-Adressen hinzufügen, von denen aus Sie sich anmelden dürfen.
3. Sie klicken **Speichern**.

## Beispiel

Sie sind in Berlin und möchten:
- Sprache auf Deutsch setzen
- Zeitzone auf "Europe/Berlin"
- Benachrichtigungen täglich um 09:00 Uhr
- Datum im Format DD.MM.YYYY

Sie öffnen Benutzereinstellungen und konfigurieren diese Optionen.

## Was passiert im Hintergrund?

Die Software speichert Ihre Einstellungen pro Benutzerkonto. Wenn Sie sich anmelden, werden diese Einstellungen automatisch geladen und die Anwendung verwendet diese Vorgaben.

## Häufige Fragen (FAQ)

**F: Wie ändere ich mein Passwort?**  
A: Im Bereich "Sicherheit" der Benutzereinstellungen können Sie Ihr Passwort ändern.

**F: Welche Zeitzonen werden unterstützt?**  
A: Es werden alle gängigen Zeitzonen unterstützt (z.B. UTC, Europe/Berlin, UTC+1).

**F: Kann ich mehrere Sprachen gleichzeitig nutzen?**  
A: Nein, Sie können jeweils eine Sprache pro Sitzung wählen.

**F: Was ist IP-Filterung?**  
A: Dies ist eine Sicherheitsmaßnahme, bei der nur Zugriffe von bestimmten IP-Adressen erlaubt sind.

**F: Wie oft werden meine Einstellungen synchronisiert?**  
A: Ihre Einstellungen werden sofort nach dem Speichern synchronisiert.
