# Kontoauszüge und Import — Ablauf für Anwender

## Voraussetzungen

Ein Konto muss vorhanden sein. Für Wertpapierimporte sollte das betroffene Wertpapier bereits angelegt sein.

## Schritt-für-Schritt-Anleitung

### 1. Datei importieren

Öffnen Sie den Importbereich und laden Sie eine oder mehrere Dateien hoch.

> **Hinweis:** Beim Massenimport müssen alle Dateien vollständig klassifizierbar sein.

### 2. Entwurf prüfen

Kontrollieren Sie die vorgeschlagenen Zuordnungen (Kontakt, Sparplan, Wertpapier, Kostenneutralität).

> **Hinweis:** Bei Warnungen zuerst die betroffenen Zeilen bearbeiten.

### 3. Verbuchen

Starten Sie die Verbuchung für den gesamten Entwurf oder einzelne Zeilen.

## Ergebnis

Die importierten Zeilen werden als reguläre Buchungen im Konto und in den abhängigen Auswertungen sichtbar.
