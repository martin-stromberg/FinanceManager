# Automatische Updates

Das Automatische Update-System (`IUpdateOrchestrator`, `UpdateExecutor`) prüft regelmäßig auf verfügbare Aktualisierungen und orchestriert deren sichere Installation auf Linux- und Windows-Produktionsumgebungen.
