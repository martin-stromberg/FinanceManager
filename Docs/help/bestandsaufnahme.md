# Bestandsaufnahme der Anwendung

Die Bestandsaufnahme wurde aus folgenden Codebereichen abgeleitet:

- UI-Routen: `FinanceManager.Web/Components/Pages` (`@page`-Routen).
- API-Oberfläche: `FinanceManager.Web/Controllers/*Controller.cs`.
- Fachmodelle: `FinanceManager.Domain/*`.
- Persistenz: `FinanceManager.Infrastructure/AppDbContext.cs`.

## Erkannte Funktionsbereiche

1. Konten und Buchungen
2. Kontoauszüge und Import
3. Kontakte
4. Sparpläne
5. Wertpapiermanagement
6. Budgetplanung
7. Berichtswesen
8. Anhänge
9. Systemverwaltung und Setup
